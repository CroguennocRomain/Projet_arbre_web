Bienvenue !

Ce site web est la finalité du projet suivant : Concevoir et développer une application de gestion du patrimoine arboré pour la ville de Saint-Quentin (Aisne).
Précédemment, nous avons analysé et nettoyé les données puis nous avons entrainé ces données sur différents modèles de machine learning. Ce site permet donc de visualiser les données et prédictions de manière simple et interactive.

Ce projet nous a permis de nous rendre compte de la relation entre les parties base de données, intelligence artificielle et développement web. Cela nous a permis de perfectionner nos compétences en les mettant en pratique dans un projet concret.
